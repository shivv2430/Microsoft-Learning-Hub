# ☁️ Microsoft Learning Hub

> **The best way to learn technology is by building with it.** — *Microsoft Learning Hub*

Learn Microsoft technologies from official resources, all in one place. Whether you are a beginner, a developer, or an entrepreneur, the **Microsoft Learning Hub** aggregates the best pathways and documentation specifically for you.

---

## ✨ Features
- **45+ Official Resources**: Carefully curated from official Microsoft platforms.
- **7 Categories**: Explore tailored resources for Developers, Startups, AI (Copilot), Data (Fabric), Low Code (Power Platform), Students, and General Skilling.
- **100% Free Learning**: Hand-picked links that guide you directly to free, official Microsoft documentation, learn modules, and tools.
- **Modern Glassmorphism UI**: Built with a stunning modern aesthetic, featuring smooth gradients and a translucent glass look.
- **Dark & Light Mode**: Seamlessly toggle between dark and light themes with preference auto-detection.
- **Real-Time Search & Filtering**: Quickly find specific tools, topics, and platforms instantly.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.

## 🛠️ Technologies Used
- **HTML5**: Semantic and accessible markup.
- **CSS3 (Vanilla)**: Flexbox, CSS Grid, Custom Properties (Variables), and Glassmorphism techniques.
- **JavaScript (Vanilla)**: DOM manipulation, event handling, real-time filtering, and localStorage (for theme preference).

## 🚀 How to Run Locally
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/shivv2430/Microsoft-Learning-Hub.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Microsoft-Learning-Hub
   ```
3. Open `index.html` in your favorite web browser, or use a local development server (like VS Code Live Server) for the best experience.
